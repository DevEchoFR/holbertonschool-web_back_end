# Python Async Function

This project introduces asynchronous programming in Python with `asyncio`.

## Files

- `0-basic_async_syntax.py`: coroutine returning a random delay.
- `1-concurrent_coroutines.py`: concurrent execution of multiple coroutines.
- `2-measure_runtime.py`: measure average runtime of concurrent execution.
- `3-tasks.py`: function that returns an `asyncio.Task`.
- `4-tasks.py`: concurrent execution using task-based helpers.
