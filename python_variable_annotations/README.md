# Python Variable Annotations

This project contains introductory exercises about Python type annotations.

## Files

- `0-add.py`: Add two floats with type annotations.
- `1-concat.py`: Concatenate two strings with type annotations.
- `2-floor.py`: Compute the floor of a float with type annotations.
- `3-to_str.py`: Convert a float to its string representation.
- `4-define_variables.py`: Define and annotate basic variables.
- `5-sum_list.py`: Sum a list of floats.
- `6-sum_mixed_list.py`: Sum a mixed list of ints and floats.
- `7-to_kv.py`: Return a tuple with a key and squared numeric value.
- `8-make_multiplier.py`: Return a multiplier function.
- `9-element_length.py`: Return elements with their lengths.
